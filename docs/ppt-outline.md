## 第2页：项目背景

**标题**: 项目背景

**内容**:
- 办公族健康问题现状
  - 久坐成疾：每天久坐超过8小时
  - 姿势不良：颈椎、腰椎问题
  - 缺乏运动：工作繁忙没时间锻炼
- 传统解决方案的不足
  - 定时提醒过于机械
  - 打断专注工作流
  - 缺乏个性化

**数据支撑**:
- 中国有超过4亿办公室人群
- 80%的办公族存在颈椎问题
- 平均每天久坐时间超过8小时

**设计建议**:
- 使用数据可视化图表
- 对比传统方案和WorkEase

---

## 第3页：用户痛点

**标题**: 用户痛点

**内容**:
- 痛点1：专注工作时被打断
  - 传统定时提醒不考虑工作状态
  - 打断深度思考，影响效率
- 痛点2：缺乏个性化指导
  - 通用建议不适合个人情况
  - 不知道该做什么运动
- 痛点3：无法量化健康状况
  - 主观感受不准确
  - 缺乏数据驱动的改善建议
- 痛点4：难以坚持健康习惯
  - 缺乏激励机制
  - 看不到长期效果

**设计建议**:
- 使用图标表示每个痛点
- 简洁的列表形式

---

## 第4页：解决方案

**标题**: 解决方案

**内容**:
- 核心思路：AI感知工作状态，智能提醒休息
- WorkSense™ 智能工作状态感知引擎
  - 实时分析用户行为数据
  - 三状态自动识别
  - 智能延迟提醒策略
- 个性化健康管理
  - AI推荐适合当前状态的动作
  - 多维度健康评分
  - 成就徽章激励

**设计建议**:
- 使用流程图展示解决方案

## 第5页：核心功能

**标题**: 核心功能

**内容**:
1. **WorkSense™ 智能状态感知**
   - 实时分析工作节奏
   - 深度专注/正常办公/休息中三状态识别
   - 智能延迟提醒

2. **AI 健康评分引擎**
   - 五维度评估：坐姿、运动、休息、作息、护眼
   - 实时评分更新
   - 个性化改善建议

3. **Smart Coach 智能教练**
   - 个性化动作推荐
   - AI推荐理由生成
   - 5分钟拉伸指导

4. **AI 健康记录**
   - 每日健康概览
   - 时间轴记录
   - 成就徽章系统

**设计建议**:

## 第6页：AI 创新点

**标题**: AI 创新点

**内容**:
- **创新点1：状态感知算法**

- **创新点2：AI 自然语言生成**
  - 动态生成状态解释文本
  - AI推荐理由生成
  - 个性化健康洞察

- **创新点3：健康收益预测**
  - 基于当前状态预测评分变化

  - 久坐风险评估

## PPT设计规范

### 配色方案

| 颜色 | 用途 |

| #10b981 | 主色调（Focus状态） |
| #3b82f6 | 辅助色（Normal状态） |
| #8b5cf6 | 辅助色（Relax状态） |
| #1f2937 | 文字颜色 |
| #ffffff | 背景颜色 |

### 字体规范

- 标题：24-32号字体，粗体
- 正文：16-18号字体
- 备注：12-14号字体

### 排版规范

- 每页内容不超过5个要点
- 使用图标辅助说明
- 保持留白，不拥挤
- 统一的动画效果
